# News Article Classification — End-to-End NLP Pipeline

End-to-end natural language processing pipeline classifying 120,000+ 
news articles into 4 categories using two different model architectures.

## Project Overview

Built as part of the AI & Machine Learning Graduate Certificate at 
Humber Polytechnic. This project demonstrates text classification 
at scale by building and comparing a traditional ML baseline against 
a deep learning approach.

**Tools:** Python · TensorFlow/Keras · Scikit-Learn · TF-IDF

## Problem Statement

Given a news article, automatically classify it into one of 4 categories:
- World
- Sports  
- Business
- Science/Technology

## Models Compared

| Model | Approach |
|-------|----------|
| Baseline | TF-IDF + Logistic Regression |
| Deep Learning | Deep Neural Network (TensorFlow/Keras) |

Both models evaluated on precision, recall, and F1-score across 
all 4 classes.

## Dataset

- 120,000+ AG News articles
- 4 classification categories
- Full train/test split with proper evaluation

## Key Skills Demonstrated

- Text preprocessing and TF-IDF vectorisation
- Multi-class classification
- Deep Neural Network design with TensorFlow/Keras
- Model evaluation — precision, recall, F1-score
- Comparing traditional ML vs deep learning approaches at scale

## Why This Matters

NLP classification is foundational to real-world AI applications — 
from ticket routing and email filtering to content moderation and 
document processing. This project demonstrates the ability to 
evaluate the right tool for the right problem rather than defaulting 
to complexity.
